* Click RunThisFirst.bat to run it and the setup wizard will help you get your computer set up for the workshop. 

* If for whatever reason this fails, ask will or sam and they will help you install it manually.

* Finally load up your new installation of MonoDevelop, click on Tools > Add-In Manager > Install from file...

* Navigate to the location of where the setup file resides and click MonoGame.mpack

* Then after a breif loading dialog, close Add-In Manager and click on Start New Solution.. > C# > MonoGame Windows OpenGL Application. Add some details about your new game below and click ok.

* Your ready to code :D. Press F5 to test.
